# GitHub Explorer

App para buscar usuarios de GitHub construida con React + TypeScript + Tailwind.

## Tech stack
- React 18 + TypeScript
- Vite
- Tailwind CSS v3

## Features
- Búsqueda con debounce (500ms)
- Perfil del usuario con stats
- Listado de repositorios propios (sin forks), ordenados por stars
- Manejo de estados: loading, error, empty

## Correr el proyecto
npm install
npm run dev
